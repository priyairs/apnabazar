ApnaBazaar is a location-based web application that helps users discover nearby vendors, small shops, and local businesses. Users can search for trusted vendors in their area, view shop details, rate and review vendors, and navigate to shop locations. Vendors can register, list their services/products, and manage their business profile on the platform.

---

## 🚀 Overview

- Find nearby local vendors and small shops using real-time geolocation
- Vendors register and list their services/products
- Secure user and vendor authentication (JWT / Firebase)
- Search and filter vendors by category, distance, and ratings
- Users can give ratings and reviews
- Map-based navigation to vendor locations (Leaflet/OSM)
- Role-based access control for users, vendors, and admin
- Fast, responsive, and scalable MERN stack architecture
- Admin panel for managing vendors and users

---

## 📌 Abstract

ApnaBazaar is a platform that connects customers with nearby street shops, vendors, and small businesses using location-based services. It helps users discover trusted vendors in their city while allowing vendors to promote their business, update details, and interact with customers. The system supports secure login, session management, map-based navigation, reviews, vendor filtering, and a personalized discovery experience.
The goal is to support local businesses, improve their digital visibility, and make vendor discovery simple and efficient for users.

## 🛠 Tech Stack

### Frontend
- React.js + Vite
- HTML5, CSS3, Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB (MongoDB Atlas)

### Maps & Location
- Leaflet + OpenStreetMap
- Browser Geolocation API

### Authentication
- JWT (JSON Web Token) / Firebase Authentication

---

## 🔁 Workflow Summary

1. User opens the web app
2. User's location is detected via Geolocation API
3. Nearby vendors are fetched from MongoDB and displayed on map
4. User searches and filters vendors by category, distance, or rating
5. User views vendor details, ratings, and reviews
6. User navigates to vendor location using Leaflet map
7. Vendors update shop info and manage profiles in real time
8. Admin monitors and manages platform activity

---

## ⭐ Description

ApnaBazaar is a MERN stack location-based web application that helps users discover nearby vendors and small businesses. It provides secure authentication, vendor listing, interactive map-based discovery, reviews and ratings, and shop navigation using Leaflet and OpenStreetMap. Vendors can manage their profiles and service listings, while users enjoy a fast, responsive, and location-aware browsing experience. The platform aims to support local businesses by giving them a digital presence and connecting them with nearby customers efficiently.
